Web TinyMCE

The default editor is just not enough

Odoo's default WYSIWYG Rich Text HTML Editor is good. It's built upon Summernote, but TinyMCE is just better.

Better HTML field WYSIWYG editor

Web TinyMCE replaces HTML fields' default editor with TinyMCE. It gives richer editing experience with great features such as:


  • font family choices
  • advanced formatting: superscript, subscript, strike-through, etc.
  • Insert Special Symbol dialog
  • advanced bullet and numbered lists
  • code snippets
  • full screen mode
  • Insert videos on forms

Try it here: https://www.tinymce.com
